## Recommendations for future students
The following are Team VibeWater's top 5 recommendations for future students taking this class next semester.

**1. Send PCBs early.**
Since this class teaches the knowledge on how to make PCBs. You must start making your subsystem PCB early, and if you send it early to the Peralta lab, you will be on time for the hardware check-off. If you are taking this class in the fall, we would suggest making sure that you are ready for check off with PCB before Thanksgiving.
<br><br>

**2. Start Github assignments early**
Since you will most probably need to submit your datasheet assignments, your team needs to make sure that you do those assignments on time and submit them without flaws. You need to make sure that you use the Canvas link submission tool, which takes a snapshot of when your website was there.
<br><br>

**3. Communicate always**
Since you will be in a team of 4, communicating about updates to the assignments is important. Make sure that you meet every week to discuss assignments and assign tasks to each other to be on the same page.
<br><br>

**4. Attend office hours**
Office hours are constructive to learn about questions you have. Since you will need to make boards communicating with each other and make team block diagrams, you might have many questions when dealing with them. TAs and professors are very helpful during the office hours.nnnnn
